Challenge 5: Accessibility Statement

Good Practices Used:

One of the best practices of accessibility that is used in this challenge is the implementation of a good semantic structure. Some users that are visually impaired rarely read the entire webpage, but scan it for interesting or relevant headings. By using heading tags, I have made it easier to identify the desired content on the webpage.

Bad Practices Used:

Due to a lack of knowledge and experience, there are some bad practices on the webpage that could be fixed. First, for challenge 2, there is no alternative text or description of the chess board for those that are visually impaired. To fix this problem, I could add a text description of the chess board. Next, the color contrast for challenge 2 and 4 have not been verified. To fix this, I could utilize a color contract checker to assess if I have used colors that are within the proper ratio. Lastly, the interactive content is missing Web Accessibility Initiative-Accessible Rich Internet Application (WAI-ARIA) content. To make the website better for the users that rely on screen readers, I should implement this interactive content.
