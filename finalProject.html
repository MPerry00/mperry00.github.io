Final Project: JavaScript-Based Game

For this assignment, we had to use our knowledge of HTML, CSS, and JavaScript to create a JavaScript-based tool, game, or other interactive feature. For my interactive feature, I created a random color generator. When the user clicks the button, a random color value is generated (seen through the text that changes next to the color). In addition, the background color of the page will change to show what that color value looks like.
